# CSharp

Command build web: cmd /c npm run dev
Website Tech: React, TypeScript, TailWind, Vite

React (v19): Thư viện JavaScript cốt lõi của giao diện. Ứng dụng sử dụng kiến trúc Component và các React Hooks (như useState) để quản lý trạng thái, ví dụ như việc chuyển đổi mượt mà giữa trang Đăng nhập và Bảng điều khiển (Dashboard).

TypeScript: Phiên bản nâng cấp của JavaScript với hệ thống định kiểu tĩnh (static typing), giúp mã nguồn an toàn, dễ bảo trì và hạn chế tối đa lỗi (bug) trong quá trình phát triển.

Tailwind CSS (v4): Framework CSS theo hướng tiện ích (utility-first). Toàn bộ giao diện từ bố cục chia đôi màn hình, màu sắc (cam #FF7F50), cho đến các hiệu ứng phức tạp như kính mờ (glassmorphism - backdrop-blur) đều được viết trực tiếp bằng Tailwind.

Vite: Công cụ build (build tool) thế hệ mới, giúp ứng dụng khởi động và biên dịch cực kỳ nhanh chóng so với các công cụ cũ như Webpack.
Lucide React: Thư viện icon mã nguồn mở tuyệt đẹp. Các biểu tượng bạn thấy trên giao diện như Micro (Mic), Giỏ hàng (ShoppingBasket), hay Cửa hàng (Store) đều được lấy từ đây dưới dạng ảnh vector (SVG) sắc nét.

FONT:
Google Fonts: Ứng dụng tích hợp trực tiếp 3 font chữ cao cấp:
Poppins & Inter: Dành cho các văn bản thông thường, mang lại cảm giác hiện đại, sạch sẽ và dễ đọc.
Playfair Display: Font chữ có chân (serif) mang nét thanh lịch, sang trọng, được dùng cho câu chào mừng và tiêu đề logo.
